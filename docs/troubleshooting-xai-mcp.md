# Troubleshooting X AI MCP 1.18.0

If X AI MCP is missing, SKS should not block. The source intelligence mode remains `context7_codex_web`.

If X AI MCP is configured but tools cannot be listed, SKS records `configured_but_unverified`. If search tools such as `search`, `query`, `web`, `retrieve`, `news`, or `grok` are detected, verified current claims require X AI evidence.

Raw X AI responses stay local-only and proof artifacts use redacted summaries.
